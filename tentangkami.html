Tentang Kami.

Perkenalkan nama saya Muhammad Fikri, Mahasiswa Pendidikan Teknik Elektronika Universitas Negeri Jakarta 2018. Saya membuat website ini untuk membantu saya untuk penelitian program sarjana di prodi Pendidikan Teknik Elektronika, website ini diharapkan bisa membantu para tenaga pendidik yaitu para guru SMKN 69 Jakarta. Dalam website ini berisikan materi mata pelajaran teknik pemrograman, mikroprosesor dan mikrokontroller untuk kelas 10, mudah - mudahan webiste ini dapat membantu para guru SMKN 69 Jakarta dalam pembelajaran.
